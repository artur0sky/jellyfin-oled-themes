## ğŸ“– About

**Jellyfin OLED Themes** is a collection of high-quality, elegant themes designed specifically for OLED displays. The primary goal is to create a deeply immersive experience by utilizing **True Black** (`#000000`) backgrounds, which not only looks stunning but also helps **save battery life** on mobile OLED devices.

Inspired by the beautiful work of [ElegantFin](https://github.com/lscambo13/ElegantFin), this project takes a technical approach to styling, leveraging modern web standards and the vibrant color palette of **Tailwind CSS v4**.

## ğŸ› ï¸ Installation

1. Open your **Jellyfin Dashboard**.
2. Navigate to **General** > **Custom CSS**.
3. Copy the `@import` code for your desired theme above.
4. Paste it into the Custom CSS box.
5. Click **Save** and refresh your page.

> **Note**: For the best experience, ensure you have enabled "Theme Song" and "Backdrops" in your user settings.
